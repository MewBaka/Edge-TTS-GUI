# Edge-TTS-GUI
简易的Edge-TTS图形界面

<img width="812" alt="image" src="https://github.com/user-attachments/assets/291d25c8-6543-4c8e-9384-acf50d97148e" />

## 功能

- ✅ 选择保存路径/文件名
- ✅ 试听语音
- ✅ 按语言分类
- ❎ 设置语速

## 使用

### 从代码开始

1. 安装库
   ```pip3 install PyQt5 edge-tts```
2. 使用
   ```python3 main.py```

### 直接使用

从 [发行版](https://github.com/MewBaka/Edge-TTS-GUI/releases) 页面进行下载
